# v0.1.0 — First Public Release

**Release date:** 2026-04-13

## Highlights

- **Deterministic forecastability triage layer** with 9 information-theoretic diagnostics (F1–F9), including:
  - Catt's Forecastability Profile + Informative Horizon Set
  - Information-theoretic limits and exploitation diagnostics
  - Predictive information learning curves
  - Spectral predictability + Largest Lyapunov Exponent (gated as experimental)
  - Entropy-based complexity triage
  - Batch triage and multi-signal ranking support
- Full AMI/pAMI foundation with horizon-aware, frequency-conditioned dependence analysis
- `run_triage()` and `run_batch_triage()` as the primary deterministic entry points
- Clean hexagonal architecture with narrow ports/protocols (SOLID compliant)
- Optional agentic narration layer (never overrides numeric results)
- CLI, HTTP API (with SSE), and MCP integration (beta/experimental)
- Comprehensive theory documentation with exact paper equations and implementation notes

## What is stable today

- Core domain APIs and deterministic scorers
- Triage diagnostics F1–F4, F6–F9
- Reproducible reporting and examples

## What is beta / experimental

- CLI and HTTP API adapters
- MCP and agent narration layer
- F5 (Largest Lyapunov Exponent) — gated behind experimental flag

## Getting started (60-second golden path)

```bash
pip install dependence-forecastability

forecastability triage --help
# or in Python:
from forecastability import run_triage
result = run_triage(signal=your_series, horizons=[1, 5, 10])
```

See `docs/golden_path.md` for the full recommended onboarding.

## Project positioning

A **deterministic pre-model triage toolkit** for deciding whether, when (which horizons),
and how a time series signal should be modeled — before investing in heavy forecasting.

Built on recent advances in information-theoretic forecastability (Catt 2026 and others)
while keeping the scientific core 100% deterministic and model-free.

## Known limitations

- Finite-sample mutual information estimation caveats
- Largest Lyapunov Exponent remains experimental
- Agent narration is interpretive only

Full details: `docs/production_readiness.md`, `docs/limitations.md`, and individual theory pages.
