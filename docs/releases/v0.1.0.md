# Release v0.1.0 Upgrade Notes

Release date: 2026-04-13

## Summary
This is the first explicit public release of the AMI -> pAMI Forecastability Analysis toolkit.
It establishes the stable/beta/experimental surface contract and a tag-driven release path.

## Highlights
- Ships the AMI + pAMI deterministic forecastability core with benchmark and report pipelines.
- Documents stability boundaries for domain APIs, CLI/API adapters, MCP tools, and agent layer.
- Adds release automation that publishes build artifacts from version tags to GitHub Releases.

## Upgrade notes
No migration required for v0.1.0.

## Compatibility
- Supported Python versions: 3.11 and 3.12.
- Packaging baseline follows Semantic Versioning and Keep a Changelog.

## Artifacts
When tag v0.1.0 is pushed, the release workflow publishes:
- Source distribution (sdist)
- Wheel distribution (wheel)
