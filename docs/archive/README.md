# Archive

This directory contains implementation-phase specification documents written during the
project build phase. They are preserved for historical context but are **not maintained**.

The authoritative source of truth for all design decisions, API contracts, and analytical
rules is the production source code in `src/forecastability/`, the active documentation in
`docs/plan/`, `docs/theory/`, and `docs/code/`, and the test suite in `tests/`.

These specifications were used to drive incremental implementation across the build phase.
The project is now complete and these documents have been superseded by the actual code.
