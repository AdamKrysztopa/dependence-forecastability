# Could Have

These items are useful extensions, but they are optional for the current phase.

## 1. Multivariate directness roadmap

Goal:
- Explore extension of pAMI-style directness diagnostics beyond a single target-plus-lag chain.

Acceptance criteria:
- Defines a narrow first multivariate use case.
- States estimator limitations before any benchmark claims.
- Avoids promoting multivariate diagnostics to default workflow without dedicated validation.

## 2. Decision-oriented report outputs

Goal:
- Add practitioner-facing outputs that translate AMI/pAMI patterns into lag-design or model-family recommendations.

Acceptance criteria:
- Recommendations are explicitly heuristic, not causal claims.
- Report language distinguishes descriptive diagnostics from tested forecasting gains.
- Output format is reusable across canonical and benchmark analyses.
